Differentiation and Integration with SymPy

Differentiation and Integration are some of the most important topics in maths. This post walks through how python can be used to solve these kinds of problems.

May 22, 2021 â€¢ 1 min read

Differentiation and Integration and the building blocks of calculus. They are usually taught in Calculus I. These two concept are heavily used in alot of fields, including machine learning.

If you don't remember, here is a high level introduction of what these two concepts are:

Differentiation (also called derivation) measures the rate of change of a function. Think of it like calculating slope in linear models. Only difference here is that you can calculate the slope at every point and for functions that are non linear. For Example: if f(x) = x**2 then, the derivative of this function is f'(x) = 2x. The derivative function is read "f prime of x".

Intergration on the other hand measures the area under the curve. For the above function, the integration or often called antiderivative will be x**3 / 3

Calculating Derivatives and Integrals is sometimes confusing so we'll use SymPy. We will start by installing the library.

Next we import everything we need.

from sympy import Symbol, diff, integrate

Since we will be calculating for x, we'll have to create a symbol for it.

x = Symbol('x')

Lets start by writing our function

f = 2*x**3+6
f
$\displaystyle 2 x^{3} + 6$

Now we calculate the derivative of the function using sympy.

f_prime = f.diff(x)
f_prime
$\displaystyle 6 x^{2}$

So now we have the derivative of our function. Lets calculate the integral.

f_int = integrate(f_prime, x)
f_int
$\displaystyle 2 x^{3}$

So this is how we calculate both the derivative and anti-derivative of a function using sympy. There are other libraries as well that work really well, e.g scipy.

Important: These aren’t actual functions. In order to use these, you need to use "lambdify" them. We’ll start by importing it. 

from sympy.utilities.lambdify import lambdify

f = lambdify(x, f)
f_prime = lambdify(x, f_prime)
f_int = lambdify(x, f_int)

print(f"The value of function at 3 is: {f(3)} and its derivative is {f_prime(3)}")

The value of function at 3 is: 60 and its derivative is 54
